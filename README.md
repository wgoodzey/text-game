This is a C++ project that is designed to load a world from a TXT file, parsing
room data such as title, description, as well as adjacent rooms.

## Usage

### Build the project
```bash
make build
```

### Rebuild the project
```bash
make rebuild
```

### Clean artifacts
```bash
make clean
```

## Output Binaries

- `build/text-game`

## Requirements

- `gcc`, `g++`
- `make`
- `cmake`
